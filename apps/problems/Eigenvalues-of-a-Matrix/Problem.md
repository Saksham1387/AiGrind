## Eigenvalues of a Matrix

Implement a function to compute the eigenvalues of a 2x2 matrix.

### Input

- **matrix** (list of list of float): The 2x2 matrix.

### Output

- **result** The eigenvalues of the matrix.

### Examples

#### Example 1

*Input:*

matrix = [[1, 2], [2, 1]]

*Output:*
[3.0, -1.0]

#### Example 2

*Input:*

matrix = [[4, 2], [1, 3]]

*Output:*
[5.0, 2.0]
